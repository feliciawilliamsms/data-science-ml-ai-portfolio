# Data Science / Machine Learning / AI Portfolio
This repository contains work samples and practice projects demonstrating applied data science,
machine learning, and AI techniques using Python and SQL.

The projects include a mix of:
- Examples inspired by real business problems encountered in professional environments.
- Self-directed projects created to strengthen skills and machine learning, NLP, and statistical analysis 
- Simulated datasets used to demonstrate approaches without exposing proprietary or confidential information
  
The goal of this portfolio is to show practical problem solving similar to what is required in industry data science and analytic roles.

---

## Skills Demonstrated 

- Python (pandas, numpy,scikit-lear)
- SQL (joins, aggregations, window functions)
- Machine Learning (classification, regression, clustering)
- Model evaluation and validation
- Feature engineering and data cleaning
- Data visualization
- Natural Language Processing (NLP, embeddings, transformers)
- A/B testing and statistical analysis
- Data exploration and reporting
- Working with real-world style datasets
